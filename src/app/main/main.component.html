Приветствую Вас, многоуважаемый Дуэлянт!

Эта страница предназначена для генерации комнаты с вашими параметрами дуэли. Для начала, введите Никнеймы дуэлянтов. Далее вам необходимо выбрать Каталог карт, на данный момент их 3:

KSF maps - все карты с проекта KSF Cybershoke maps - все карты с проекта Cybershoke KSF Bonuses Cybershoke Bonuses - все бонусы с проекта Cybershoke

Напротив каждого из пунктов есть кнопка ссылка, которая ведет на гугл-таблицу, где вы можете ознакомиться с полным перечнем всех карт определенного каталога.

Далее вам необходимо выбрать Tier (сложность) карт, которые будут участвовать в рандомайзере. Для Bonuses выбор отличается, их всего два: T1-T4 и T5-T8 Карты 1-го тира самые легкие, 8 - самые сложные. Доступен выбор нескольких позиций.

После выбора сложности карт, вам потребуется выбрать стили карт. На данный момент есть на выбор 3 стиля: Tech - Технически сложные карты, требующие знания определенных механик, умения проходить спины (крутилки), здесь больше важны реакция и понимание контроля модели в воздухе. Unit - Карты требующего полного контроля, здесь важны приземления и набор максимальной скорости. Для каталога карт Bonuses доступен 3 стиль Bhop - это бонусы с большим количеством блоков для прыжков, как правило на таких бонусах очень мало серфа.

Доступен выбор нескольких позиций.

После того как все параметры выбраны, на ваш выбор предоставляются вариации формата дуэлей: Duels - Стандарт проведения дуэлей на моем канале. Рандом 9 карт по ранее выбранным параметрам. Далее игроки по очереди (первенство также определяется рандомом) вычеркивают по 3 карты. Далее наступает стадия пиков (выбора) карт. И decider определяется автоматически. Tournament BO3 - Стандарт проведения матчей турнира в формате BO3. Рандом 8 карт. Tournament BO5 - Стандарт проведения матчей турнира в формате BO3. Рандом 16 карт.

После генерации комнаты, у вас будет возможность пересоздать лобби с новыми картами с теми же параметрами, однако для этого нужно, чтобы оба игрока нажали кнопку Reroll. Если обоих участников все устраивает. Создателю лобби необходимо нажать на кнопку Show links и отправить одну из ссылок вашему оппоненту, а самому перейти по другой.

Гладких рамп и чистых влетов, многоуважаемые Дуэлянты!
